# 2026-07-13 — Session 3.2: Paste-a-link door

## What was done

- **Importer client** (`web/src/features/patterns/importerClient.ts`): typed wrappers for
  `POST /import/extract` and `POST /import/image` over the same-origin proxy — raw PB token in
  `Authorization` (set by hand; the SDK doesn't attach it to plain fetch), 12 s
  `AbortSignal.timeout` (server budget is 8 s; this catches a hung proxy), `ImporterError`
  carrying the server error code, defensive JSON parse for a dead importer's non-JSON 502,
  og:image bytes wrapped into a `File` for the §8 pipeline.
- **Bridge** (`pendingUrlImport.ts`): one-shot in-memory stash mirroring
  `pendingAttachmentImport` (module var + UNSET-symbol ref consume, StrictMode-safe) carrying
  `{defaults, importedFrom, thumbnail}`.
- **Door hook** (`usePasteLinkDoor.ts`): `clipboard.readText()` as the first statement of the
  tap handler (iOS gesture requirement), URL sanitizer (trim, whitespace/2048 rejects,
  `https://` prefix for scheme-less text, http/https + hostname-dot checks), extract → optional
  image proxy → `processImage` (any image failure soft-fails to no thumbnail), stash → navigate
  → toast. Non-link clipboard stays inline on the door; every server failure takes the soft
  path (form opens with only the URL filled + gentle `info` toast). `busyRef` double-tap guard;
  `mountedRef` discards a late stash and revokes its preview URL.
- **Doors**: third lilac door (ClipboardPaste icon) added FIRST in `QuickAddSheet.tsx` and
  `DoorsRow.tsx` (now `grid-cols-3`), per DESIGN's order; dual-busy composition disables all
  doors + the sheet backdrop while either hook runs.
- **Form landing**: `PatternFormPage.tsx` consumes the URL bridge beside the file one (file
  door wins if both; they can't coexist) and merges `defaults` over `patternFormDefaults`;
  `PatternForm.tsx` gained `importedFrom` → lilac "imported from {site}" chip below the
  thumbnail field. Mapping: title←og:title, designer←meta author, source_url←canonical_url,
  source_name←og:site_name (fallback: hostname sans www), notes←escaped og:description `<p>`.
- `web/package.json` → 0.3.2 · four DECISIONS.md lines · plan archived as
  `logs/p10-session-3.2-paste-link-plan.md`.
