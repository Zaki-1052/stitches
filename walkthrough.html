Logging in

The login screen — /login
Open the app — you should see a yarn ball, the "stitches" wordmark, and "invite-only" at the bottom No sign-up link exists — that's by design
Try a wrong password — you get a gentle inline error, no scary red box
Log in with a seeded account — lands on the home screen Cece or Friend demo account from the seed script
Home tab

Home — "On the Hook"

Dock → Home (first icon) — /
Greeting + avatar at the top — tap the avatar to go to Settings
Hero card(s) — your in-progress project(s) with cover photo, name, live counter value, and progress bar Multiple WIPs? Swipe sideways to snap-scroll between them
Tap "Count" on a hero card — jumps straight to the full-screen counter
Tap "Journal" on a hero card — opens a quick journal-entry sheet (write something, add a photo)
Three "quick add" doors — Paste a link, Add a file, Type it in Each one opens the pattern form in a different way
"Recently added" strip at the bottom — horizontal scroll of your newest patterns Finished patterns show a mint "made ✓" badge
Empty state — if the account has no patterns/projects, you see a yarn-basket illustration + "start your library" Try with a fresh account if you can
Library tab

Pattern Library

Dock → Library (second icon) — /patterns
Grid view — 2-column grid of pattern cards with thumbnails, titles, designer, tag dots
Toggle to list view — compact rows with small thumbnails (icon in the top area)
Search — type in the sticky search bar; filters by title and designer Try searching for something that doesn't exist — distinct "no results" message
Filter chips — tap to open a filter sheet (shelf, craft, weight, tags) Filters stick in the URL — refresh the page and they're still there
"Made ✓" badge — patterns with a finished project show a mint badge
Patterns

Creating & Viewing Patterns

Dock ➕ or Library → tap a pattern
Tap the ➕ button in the dock — opens a quick-add sheet with three doors Paste a link · Add a file · Type it in
"Type it in" door — opens a blank create form with title, thumbnail upload, craft, shelf, and a collapsible details section
Upload a photo from camera roll — HEIC converts automatically, image is compressed to WebP
Hook field — enter a mm number and see the US letter alias live e.g. 5.0 mm · H-8
Tags — create tags inline, pick a patch color (blue, coral, lilac, mint, butter)
Rich-text notes — the notes field is a real editor (bold, italic, lists)
Save — sticky bar at the bottom, then land on the pattern detail page
Pattern detail — hero image, shelf toggle, meta chips (craft, CYC weight, hook, gauge, difficulty), tags, notes, photos
Visibility toggle — switch between "Private" and "Friends" sharing "Friends can see this pattern's info and photos — never your files."
Delete a pattern — plain-language confirm; blocked if it has linked projects (offers to unlink)
Vault

Attachments — the Copyright Vault

Pattern detail → Attachments card (owner only)
Attachments card — lock icon + "Only you can ever see these" This card is completely invisible if you're viewing someone else's shared pattern
Upload a PDF — page 1 becomes the pattern thumbnail automatically if there isn't one yet
"Add a file" door — from ➕ or Home, pick a PDF/scan → the form opens pre-filled with a thumbnail from the first page This is the "file-first" import: upload a PDF and get a pattern + attachment in one flow
Pattern text — type or paste pattern instructions; stored owner-only like files
Protected files — View links use short-lived tokens; a leaked URL is useless
Projects tab

Projects — the Craft Journal

Dock → Projects (fourth icon) — /projects
Projects list — cards grouped by status; "In progress" first. Each shows cover, name, pattern, status chip, progress bar
Status chip filter — tap to show only one status
Create a project — from the projects page or from a pattern detail ("Start a project") Can optionally link to a pattern, or start an improvised make
Five statuses planned in progress finished frogged hibernating
Flip to "Finished" — star confetti in all five patch colors, finished date defaults to today, and a prompt offers a "finished photo" journal entry Confetti plays once, then never again for that project
Reduced motion — turn on "Reduce Motion" in iOS settings → no confetti, everything still works
Frogged status — "Frogged — rip-it happens." microcopy
Empty state — "Nothing on the hook. Chain on!"
Detail

Inside a Project

Projects → tap a project card
Cover photo + status chip — tap the chip to change status via a bottom sheet
Pinned summary — edit in place, it's the "state of the project" note
Counters card — rows with labels, values, ± buttons, link glyphs when linked Tap "Open counter" for the full-screen counting experience
Create counters — add label, optional target, optional "resets with" link to another counter e.g. Stitch counter resets when Rows increments
Journal feed — date-grouped entries with photos, composer at top ("Add an entry… 📷")
Write a journal entry — add text and/or photos, editable date for backdating Backdated entries sort into the right date group
Edit / delete entries — each entry has edit and delete actions
Flagship

Counter Screen

Project detail → "Open counter" — /projects/:id/count
Full-screen counter — giant number, big +1 tap zone at the bottom (reachable by either thumb)
Tap tap tap — the number does a soft roll-pop animation and the background pulses blue
−1 / edit value / reset — row above the tap zone with these three actions
Progress bar — "12 of 48 rows" when a target is set
Target reached — one mint star pops when you hit the target number
Multiple counters — swipe or tap the strip below the number to switch counters Dots show which counter is active
Linked counters — if Stitches "resets with" Rows, incrementing Rows zeroes Stitches A caption explains the link: "Rows +1 resets Stitches"
Dim mode 🌙 — tap the moon icon for a dark blue night theme (couch crochet mode) The number gets a soft blue glow; preference persists
Wake lock ☕ — tap the coffee icon to keep the screen on; re-acquires if you tab away and back
Offline resilience — turn on airplane mode, tap +1 a bunch, kill the tab, reopen, turn off airplane mode → server lands on the exact value A tiny stitch icon appears while taps are queued to sync
Two-browser sync — open the same project in two browsers; taps sync live both ways via realtime
Import

Paste-a-Link Import

➕ or Home → "Paste a link"
Copy a pattern URL to your clipboard (Ravelry, a craft blog, anything)
Tap "Paste a link" — iOS asks for clipboard permission, then it extracts the page title, author, site name, and thumbnail Shows a lilac "imported from {site}" chip on the form
Pre-filled form — title, designer, source URL, source name, thumbnail all land from the page's metadata Enrich and save — you're adding to the library, not just bookmarking
Soft failure — paste a garbage URL → form opens with just the URL filled + a gentle toast Manual entry is never blocked by a bad scrape
Settings

Settings

Home → tap avatar — /settings
Profile — change name + avatar (image goes through the compression pipeline) Header updates immediately after saving
Change password — old + new + confirm (matters: there's no email reset) Wrong old password gives an inline error
Counter preferences — wake-lock default, dim-mode memory, experimental haptic-tick toggle Haptic tick needs iOS 17.4+ and the toggle defaults to off
Sign out — and version number at the bottom
